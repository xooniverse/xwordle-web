Privacy Statement for Wordle Bot

Date of Last Revision: 4 July, 2024

Introduction

Welcome to Wordle Bot, a Telegram bot version of the popular Wordle game. Wordle Bot is created by @Xooniverse with ❤️ for Telegram users. The game involves guessing a five-letter word with six tries, where hints are provided after each guess about the letters' correctness.

Wordle Bot Overview

Wordle Bot brings the fun of the Wordle game (available at The New York Times) to Telegram. Our bot is open source, and you can find the source code on GitHub: GitHub Repository.

Technology and Dependencies

The bot is developed entirely in Dart, using Televerse (a Telegram Bot Framework for Dart, also developed by our team). We utilize the Free Dictionary API to validate words and Firebase Firestore as our database.

Data Collection

To provide an engaging and personalized gaming experience, we collect certain data:

  • Telegram API Data: Basic data provided by the Telegram Bot API, including:
    • User’s Telegram Account’s First Name
    • User’s unique Telegram ID (an integer)
  • Game-specific Data, including but not limited to::
    • Number of games played by the user
    • User’s win streak
    • Number of wins and losses

Personal Identifiable Information

We do not collect any personally identifiable information other than the first name (which may or may not be the user's actual name) and the unique Telegram user ID.
